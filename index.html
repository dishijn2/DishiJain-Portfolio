The Bin Packing Problem

Data Structure and Algorithms • Dec 2018 - March 2019

The BIN PACKING PROBLEM concentrates on placing of objects subject to various constraints (example length, breadth, width etc.) into a finite number of bins, minimalizing the space wastage. The arrangement of bins is done keeping in mind the fact that one unit of a kind of objects, stays together and is not further split. The algorithm developed finds applications such as loading containers, filling trucks etc. The problem is an example of an NP-hard problem; hence algorithms are to be developed keeping in mind to provide if not perfect, optimal solution.
